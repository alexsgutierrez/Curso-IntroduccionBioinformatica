# Curso-Introduccion-a-la-bioinformatica
Este repositorio forma parte del curso "Introducción a la bioinformática para el análisis de datos de secuenciación Illumina". El objetivo es familiarizarse con los comandos básicos y programas comúnmente utilizados en el procesamiento bioinformático para la obtención de SNPs a partir de secuenciación genómica en Illumina.
El curso tiene diferentes módulos:
1. Introducción a los comandos básicos en Linux
2. Control de calidad
3. Alineamiento
4. Búsqueda de SNPs
